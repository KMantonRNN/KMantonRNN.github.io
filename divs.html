Divs and Containers

Page structure and segmentation rules

HTML structure is based on two concepts; the first being cascade rendering. Wherein the order the elements appear in the HTML document dictates the order they appear on the page. Elements that are added first are rendered first. The second concept is DIV elements. These are invisible elements used to aid in structure - almost like a box. They can be manipulated to be any size and shape and can be arranged on a page.

Below is a wireframe of a sample website. Hover over named elements to see the individual divs and see the code to reproduce it.
